Modulo Software de diseño

🎨 Submódulo: Diseño Digital

"Manipulación Fotográfica y Composición Visual Creativa" Este submódulo se centró en el dominio de Adobe Photoshop, el estándar de la industria para la edición de imágenes rasterizadas. El objetivo fue desarrollar el "ojo clínico" para el diseño, aprendiendo a mejorar, transformar y crear piezas visuales desde cero o a partir de fotografías existentes.Al principio, la interfaz puede abrumar, pero aquí aprendí los pilares que sostienen cualquier diseño: Gestión de Capas (Layers): Entender la jerarquía visual, cómo apilar elementos y usar la visibilidad para organizar un proyecto complejo. Formatos y Resoluciones: La diferencia crítica entre DPI (para impresión) y PPI (para web), además de manejar formatos como PSD, PNG y JPG. Herramientas de Selección: El uso de lazo magnético, varita mágica y la pluma para aislar objetos con precisión quirúrgica.Aquí es donde aprendí a "curar" imágenes y mejorar la realidad: Herramientas de Clonación y Parche: Uso del Tampon de clonar y el Pincel corrector puntual para eliminar imperfecciones, personas u objetos no deseados de una foto. Ajustes de Color y Exposición: Manipulación de niveles, curvas, brillo y saturación para cambiar por completo la atmósfera de una imagen. Filtros Inteligentes: Aplicación de desenfoques, enfoques y licuado para estilizar retratos o paisajes sin destruir la imagen original.

🌐 Submódulo: Páginas Web

"Arquitectura de Contenido y Maquetación Estándar" Este submódulo se centró en el dominio de HTML5 (HyperText Markup Language) como el lenguaje de marcado fundamental de la red. El objetivo fue entender que una página web no es un dibujo, sino una estructura lógica de etiquetas que el navegador interpreta para mostrar información de forma jerárquica y accesible.Una web sin enlaces o imágenes es solo un documento aburrido. En esta etapa dominé: Hipervínculos (a): Creación de enlaces internos entre páginas del mismo sitio y enlaces externos a otras webs. Multimedia: Inserción de imágenes con img, videos con video y audios, aprendiendo siempre a usar el atributo alt para la accesibilidad. Listas y Tablas: Organización de datos mediante listas ordenadas (ol), desordenadas (ul) y tablas complejas para mostrar información estructurada.El uso de VS Code cambió por completo la experiencia de escribir código: Emmet: El uso de abreviaciones para escribir bloques de código gigantes con solo unas pocas teclas (ej. escribir ! y dar tab para crear toda la estructura base). Extensiones: Instalación de herramientas como Live Server para ver los cambios en el navegador en tiempo real cada vez que guardaba el archivo. Depuración Visual: El uso de colores en la sintaxis para detectar rápidamente si olvidé cerrar una etiqueta o si cometí un error de escritura.

Reflexión: Del Estrés a la Fascinación

"Escribir HTML puro fue, al principio, una experiencia muy estresante. Ver una pantalla llena de etiquetas como y s puede ser abrumador, y un solo cierre de etiqueta olvidado podía arruinar toda la página. Sin embargo, hay un punto de inflexión: cuando logras que la estructura funcione y ves tu creación en el navegador, el código te atrapa. Se vuelve casi adictivo ver cómo tus líneas de texto se transforman en una interfaz real. VS Code fue mi mejor aliado en este proceso, convirtiendo el caos inicial en una pasión por construir cosas para la web."
