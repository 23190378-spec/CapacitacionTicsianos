Modulo Software Aplicacion

📱 Submódulo: Aplicaciones Informáticas

"Del Pensamiento Lógico a la Realidad Móvil" Este submódulo se centró en el ciclo de vida del desarrollo de software para dispositivos móviles, utilizando MIT App Inventor como plataforma principal. El objetivo no fue solo "hacer apps", sino entender la lógica de programación detrás de cada interacción y la importancia de la experiencia del usuario (UX).

En lugar de escribir líneas de código complejas, el aprendizaje se basó en la programación orientada a eventos mediante bloques. Lógica de bloques: Aprendí a estructurar algoritmos utilizando condicionales (si... entonces), bucles (por cada...) y variables globales para gestionar datos dentro de la aplicación. Gestión de Componentes: Aprendí a diferenciar entre componentes visibles (botones, imágenes, lienzos) y no visibles (sensores de acelerómetro, bases de datos locales TinyDB, motores de texto a voz).

Uno de los pilares de mis proyectos fue la creatividad visual. No me limité a los botones estándar; exploré: Psicología del color: Selección de paletas atractivas y coherentes. Interactividad: Uso de lienzos (Canvas) y animaciones para que el usuario sienta una respuesta inmediata al tocar la pantalla. Navegación Intuitiva: Diseño de menús y transiciones de pantalla que facilitan el uso de la app sin necesidad de un manual.

📄 Submódulo: Documentos Digitales

"Arquitectura y Gestión de Información de Alto Impacto" Este submódulo se centró en la creación de documentos de complejidad profesional. El objetivo fue transformar el procesador de textos en una herramienta de ingeniería documental, garantizando que archivos de gran extensión mantuvieran la coherencia visual y estructural..

El mayor reto fue aprender a manejar documentos extensos. Aquí aprendí que un documento profesional no se hace "a mano", sino con automatización: Estilos de Título: Implementación de niveles jerárquicos (Título 1, 2, 3) para generar Índices y Tablas de Contenido automáticas. Saltos de Sección: El uso de saltos de sección de página siguiente para permitir diferentes orientaciones (vertical/horizontal) o numeraciones dentro del mismo archivo. Panel de Navegación: Herramienta clave para moverme rápidamente entre cientos de páginas de información sin perder el hilo.

Reflexion

"Siendo honesto, este submódulo fue una prueba de paciencia. Trabajar con documentos tan cargados de información, imágenes y formatos específicos me generó niveles altos de estrés; un pequeño error en un salto de sección podía mover todo el archivo. Sin embargo, hoy estoy agradecido: aprendí que la prolijidad técnica es lo que diferencia a un aficionado de un profesional. Dominar estas herramientas me dio la disciplina necesaria para enfrentar cualquier reto administrativo futuro."

Opinion

Para mi fue divertido y estresante estos submodulos pero fue una buena introduccion ala capacitacion no me arrepiento de haber entrado ala capacitacion TICS
